 Evaluación Final – Módulo 4

 Descripción

Aplicación web desarrollada en JavaScript que consume información desde una API REST utilizando el método Fetch.

La aplicación almacena los datos dentro de una clase y permite consultar información mediante botones HTML y eventos del DOM.

 Objetivos

* Consumir datos desde una API externa.
* Almacenar la información en una clase JavaScript.
* Implementar métodos de búsqueda y listado.
* Utilizar programación orientada a objetos.
* Aplicar eventos y manipulación del DOM.

 Funcionalidades

* Listar nombres de usuarios.
* Buscar información básica por nombre.
* Mostrar dirección completa.
* Mostrar información avanzada.
* Listar compañías y frases.
* Ordenar usuarios alfabéticamente.

 Tecnologías utilizadas

* HTML5
* JavaScript
* Fetch API
* Bootstrap
